# React + TypeScript Demo

一个简单的 React + TypeScript 待办事项应用。

## 功能特性

- ✅ 添加新任务
- ✅ 标记任务完成/未完成
- ✅ 删除任务
- ✅ 任务筛选（全部/待办/已完成）
- ✅ 任务统计
- ✅ 响应式设计

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS

## 安装和运行

```bash
# 克隆仓库
git clone https://github.com/yqq-a/demo.git
cd demo

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

```
demo/
├── src/
│   ├── App.tsx       # 主应用组件
│   ├── main.tsx      # 应用入口
│   └── index.css     # 全局样式
├── index.html        # HTML 模板
├── package.json      # 项目配置
├── tsconfig.json     # TypeScript 配置
├── vite.config.ts    # Vite 配置
└── tailwind.config.js # Tailwind CSS 配置
```
